# Jestore Admin

Panel administrativo demo para gestionar pedidos, productos y clientes.

## Ejecutar localmente

```bash
cd app
npm install
npm start
```

La app queda disponible en `http://localhost:3000`.

## Desplegar en Render

El proyecto incluye `app/render.yaml`.

En Render usa:

- Root Directory: `app`
- Build Command: `npm install`
- Start Command: `npm start`
